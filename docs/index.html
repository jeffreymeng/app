Intro

The Jeffrey Viewer API provides a simple URL-based file viewing experience. It supports many formats and provides the user with a beautiful interface to view them. After viewing the file, the page will link back to your url.

Api Limit

There is currently no API limit, though if you are going to use the API heavily, please contact us first - viewer jeffkmeng.com.

Getting Started

All file types utilize the same url, referred to as an open URL:

https://viewer.jeffkmeng.com/open

The open URL(above) is the base of the path you will redirect to, regardless of parameters, but this is most likely not going to be the URL your users see. All URL's have base and type-specific file parameters. Base file parameters apply to all file types, while type-specific parameters apply only to a certain data type. There are required base and type-specific file parameters, as well as optional parameters for both.
